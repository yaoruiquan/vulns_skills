|------|------|

| 元素 | 说明 |

## 表单字段

**注意**：以下选择器需通过 `take_snapshot` 获取实际 uid，CNNVD 表单结构待实测后补充。

| 字段 | 类型 | 说明 |
|------|------|------|
| 漏洞名称 | textbox | 漏洞标题 |
| 漏洞类型 | combobox | 下拉选择 |
| 影响产品 | textbox | 受影响产品名称 |
| 影响版本 | textbox | 受影响版本 |
| 受影响实体描述 | textbox/multiline | 需联网搜索获取 |
| 验证过程 | textbox/multiline | 复制 docx 内容 |
| 提交者单位 | textbox | 固定值 |
| 技术支持电话 | textbox | 优先 docx，否则 15700082275 |
| 验证码 | textbox | 需人工或 OCR 识别 |
| 提交按钮 | button | 提交表单 |
